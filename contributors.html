Analogies

Each analogy is described in a single file, in the _nms folder.

To add a new analogy, just add a file to that folder. Follow our naming conventions and make sure to provide all the necessary frontmatter.

Each analogy is accompanied by a picture. The pictures are stored in the assets/images/nm folder. Make sure the picture has a square aspect ratio (crop it before adding it). Ideally pictures should have a colorful background (this leads to a more pleasing look of the corresponding card).
